## 📌 Sobre o projeto

Este projeto foi desenvolvido para praticar desenvolvimento **Front-End**, criando um site moderno e responsivo sobre o piloto brasileiro **Gabriel Bortoleto**, que compete na Fórmula 1.
Meu primeiro projeto com JavaScript, apenas para fins de estudos.

Estou com dificulades e ainda não consegui manter ele todo configurado pra mobile, se alguém tiver alguma sugestão e quiser ajudar mantendo o mesmo layout e fazendo pequenas alterações... 

O site inclui:

- Página inicial com design imersivo
- Página com informações sobre a carreira do piloto
- Página com a próxima corrida
- Animações em JavaScript
